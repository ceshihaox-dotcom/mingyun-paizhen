# 命运牌阵 · Tirage du Destin

一个静态网页的「穿越体验」抽卡工具：抽一个时空坐标，配上母题 / 身份 / 变数三枚骰子，组成一次穿越设定。

线上：https://mingyun-paizhen.vercel.app

## 页面

- **`index.html` — 抽卡**：抽一张时空坐标 + 母题 / 身份 / 变数，可按题材（日常 / 混合 / 冲突 / NSFW）多选筛选。
- **`牌库.html` — 牌库索引册**：全部时空坐标，按大区 / 时代浏览，支持时间线视图、搜索、筛选；自带增 / 改 / 屏蔽（存本地）。
- **`维度.html` — 其他维度**：母题 / 身份 / 变数的管理（屏蔽 / 添加）。

## 本地运行

纯静态站点，任意静态服务器指向仓库根目录即可。仓库自带一个无缓存预览服务：

```
node _serve.js      # 默认 http://localhost:8788
```

## 数据

`data.js` 是全部词条，挂在 `window.CK_DATA` 上。时空坐标字段：名称 / 拉丁 / 大区 / 国家 / 时代 / 年代 / 时段 / 基调 / NSFW / 主题 / 氛围 / 感官锚点 / 备注。

`data.js` 是构建产物（由独立的源数据项目生成），本仓直接提交、可直接编辑。

## 自定义

`ck-custom.js` 是用户自定义层：屏蔽 / 编辑 / 添加都按**词条名**存进 localStorage，与默认数据分离——更新默认数据不会覆盖你的自定义。无硬删除。
